## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Run and Watch Unit Tests for Development

```sh
pnpm test:unit-dev
```

### Run Unit Tests for Production

```sh
pnpm test:unit
```

### Run E2E Tests

```sh
pnpm test:e2e
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```
